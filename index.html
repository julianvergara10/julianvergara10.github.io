Simple File Upload Form

This form uploads a file to a spreadsheet and shared folder connected to this project.

It only took a few simple steps:

  1. Set the form action to the project's API endpoint
  2. Set the form's enctype attribute to multipart/form-data. This tells the browser to send a file
  3. Add a file input button
  4. Create columns in your spreadsheet matching the names of the form elements

Click Publish to find out how to put this widget on your site.
